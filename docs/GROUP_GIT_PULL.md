# Group Git Pull

Repo we are using:

https://github.com/JaeyeongSeol/DAN

To get the project:

```bash
git clone https://github.com/JaeyeongSeol/DAN.git
cd DAN
```

To update your copy before working:

```bash
git checkout main
git pull origin main
```

Quick reminder: do this before starting a task so your branch is based on the newest group work.
